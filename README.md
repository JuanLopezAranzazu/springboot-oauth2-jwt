# Spring Boot + Oauth2 + Jwt

Este es un proyecto de una API REST con Spring Boot usando PostgreSQL, Oauth2 y Jwt para autenticación de usuarios.

## Tecnologías Utilizadas
- Java 17+
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok
- Spring Security
- Oauth2 Client
- Jwt

### Construir y ejecutar
```sh
mvn clean install
mvn spring-boot:run
```
